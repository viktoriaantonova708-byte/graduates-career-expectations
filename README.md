Опросник: Карьерные ожидания выпускников

Одностраничное веб-приложение для сбора и анализа данных о карьерных предпочтениях студентов и выпускников.



Технологии

\- Python 3.12

\- Streamlit (Веб-интерфейс)

\- Firebase Firestore (Облачная NoSQL база данных)

\- Pandas \& Plotly (Анализ данных и визуализация)



Запуск локально

1\. Клонируйте репозиторий.

2\. Установите зависимости: `pip install -r requirements.txt`

3\. Получите `serviceAccountKey.json` из консоли Firebase и поместите в корень проекта.

4\. Запустите: `streamlit run app.py`



Выводы по собранным данным (Пример)

Большинство опрошенных (70%) предпочитают гибридный формат или удалёнку.

Главным приоритетом выпускники считают не зарплату, а возможности карьерного роста и обучения.

Основные переживания выпускников состоят в конкуренции и в отсутствии опыта
